# AWAM



## Introduction
AWAM: AN ADAPTIVE WAVELET ATTENTION MODEL FOR METEOROLOGICAL SUPER-RESOLUTION PREDICTION

## Installation
This project provided an environment seeting file of conda,users can easily reprodece the environment by the following commands
```
conda env create -f environment.yml
conda activate AWAM
```

## train and test 

To training or testing, use 
```
python main.py
```
